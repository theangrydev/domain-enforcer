# domain-enforcer
Enforce dependencies between packages in a unit test!

```xml
<dependency>
    <groupId>io.github.theangrydev</groupId>
    <artifactId>domain-enforcer</artifactId>
    <version>1.1.0</version>
</dependency>
```
